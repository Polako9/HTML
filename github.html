GITHUB is a website where you can put your code to the various programming language. It is working as your online portfolio but some of the repositories can be private if you dont want show them to bigger audience.
When you send changes from your working computer to the repository- you make commit(it is something like history of your action with server), you can also work base on branching and together with your firends make application without worry about someone destroy your code.
We can download application for desktop to easly work with GITHUB website. Everything can be send on server by using graphic interface or you can use terminal to make it faster.

If you want upload your file to GITHUB by using terminal just make this steps bellow:

  1. Create dir on your computer using comend "cd"
  2. Make a repo on your github account
  3. Copy your repository link
  4. type git init and press enter
  5. then git add .
  6. then git commit -m "text"
  7. then git remote add origin https://github.com/Polako9/HTML.git
  8. in my face i use git push --force origin master because i make readme file in other case just type git push -u origin master if you dont have any file to download from your repo
